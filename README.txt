OBJETIVOS DEL PROYECTO

Objetivo General

-Desarrollar un Software educativo que permita conocer e identificar la diversidad biológica Colombiana continental y marítima


Objetivos Específicos

- Concientizar a los miembros de los planteles educativos acerca de los peligros y amenazas que afectan la flora y fauna de la región

- Conservar las áreas de especial importancia ecológica,


FUNCIONALIDAD EL PROGRAMA

La funcionalidad del programa esta enfocada en brindar información diferentes tipos de especies animales y vegetales, esto se logra a través de una entrada de usuario en donde se digita el nombre común o vernáculo de un animal o planta y mediante una API se determina el conjunto de datos que la aplicación utilizará para imprimir en la consola los datos requeridos. En este caso los datos a mostrar son:

1. Nombre vernáculo
2. Nombre científico
3. Reino
4. Familia
5. Estado de conservación
               

HERRAMIENTAS UTILIZADAS

- Python
- Librería requests: Se utiliza para hacer solicitudes HTTP a la API de datos.gov.co.
- Fuente de datos: La API del recurso "Listado de Especies Amenazadas" disponible en datos.gov.co 


CONCLUSIONES

Este mini proyecto se hizo con el fin de dar a conocer información referente a los riesgos que inciden en la biodiversidad de la región, para ello se utilizó una Api publica con los datos necesarios para elaborar una herramienta educativa e informativa, siendo una base inicial útil para futuros proyectos mas complejos en donde se implementen graficas y sistemas de análisis de datos.
